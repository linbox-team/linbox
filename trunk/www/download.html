Developer releases

N.B. The releases in this section are developer releases! They are not guaranteed to be fully functional or bug-free. Please do not use them to prove key theorems and subsequently publish the results in a refereed journal. Doing so may result in being tarred and/or feathered!

We would, however, very much like feedback and contributions. Please give a developer release a try if you are feeling bold.

Developer releases are currently not available, as the Stable release is newer. Please use that instead. Thanks!
